Ovine Brain MRI Atlas

Corriedale Ovine Brain Atlas for use in Atlas-Based Segmentation

Sheep MRI Brain Atlas Files

  • High Resolution (7T) Template
  • Low Resolution (1.5T) Template
  • Skulled Template
  • Automatic Segmentation Script
  • Template Labels Combined
  • Template Labels Individuals
  • Template Mask (for deskulling)

MRI Template Details

Low Resolution Sheep Atlas created using ANTS for template creation and ITK-SNAP for labeling. Subjects were 13 Adult Corriedale Ewe Brain MRIs. Labels are the Left and Right Motor Cortices, Ventricles, Thalamus, Hippocampus, Caudate, Amygdala and Cerebellum.

How to use this Atlas

The easiest method of utilising this atlas is to run the automatic segmentation script with the necessary arguments. The script utilises FSLs FLIRT and ANTS SyN Greedy Algorithm, both programs are required for the algorithm to work.

Alternatively, one can register the low resolution template to a subject to obtain a translation vector. The translation vector can be applied to the template labels to obtain translated labels. The deskulling template and template mask can be used to perform automatic deskulling.
