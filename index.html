MRI Template Details

Low Resolution Sheep Atlas created using ANTS for template creation and ITK-SNAP for labeling. Subjects were 13 Adult Corriedale Ewe Brain MRIs.

How to use this Atlas

The easiest method of utilising this atlas is to run the automatic segmentation script with the necessary arguments. The script utilises FSLs FLIRT and ANTS SyN Greedy Algorithm so both programs are required for the algorithm to work.

Alternatively, one can use the low resolution template and register it to a subject. The translation algorithm can then be applied to the labels to obtain translated labels.
